Lenguajes Utilizados:

Casa de Cambio

Casa de Cambio es una aplicación web que permite a los usuarios convertir diferentes monedas. Los usuarios pueden seleccionar la moneda de origen, el tipo de operación (compra o venta) y la cantidad que desean convertir. Al hacer clic en el botón "Convertir Moneda", se muestra el resultado de la conversión.

Lenguajes Utilizados:

  • ✔️ HTML: Para la estructura básica de la página.
  • ✔️ CSS: Utilizando Milligram para un diseño limpio y responsivo, además de un archivo CSS personalizado (estilo.css) para estilos adicionales.
  • ✔️ JavaScript: Para manejar la lógica de conversión de monedas y la interacción del usuario.
  • ✔️ SweetAlert: Para mostrar alertas atractivas y mejorar la experiencia del usuario.

Desafíos Superados:

  • Interactividad: Implementar la lógica para convertir monedas en función de la selección del usuario y la cantidad ingresada.
  • Diseño Responsivo: Asegurarse de que la aplicación se vea bien en diferentes dispositivos y tamaños de pantalla.
  • Manejo de Errores: Implementar validaciones para asegurar que los usuarios ingresen datos válidos y manejar situaciones como la conversión de cantidades no numéricas.

Adivina el Número

Adivina el número es un juego interactivo donde los usuarios intentan adivinar un número entre 1 y 10. La idea es simple y divertida, ideal para practicar habilidades de lógica y adivinanza.

Lenguajes Utilizados:

  • ✔️ HTML: Para la estructura básica de la página.
  • ✔️ CSS: Se utiliza Milligram para un diseño limpio y moderno, además de un archivo CSS personalizado para estilos adicionales.
  • ✔️ JavaScript: Para la lógica del juego, donde se manejan las interacciones del usuario y se muestran los resultados.
  • ✔️ Axios: Incluido en el proyecto, aunque no utilizado en esta versión, pero podría ser útil para futuras expansiones.

Desafíos Superados:

  • Interactividad: Implementar la lógica para que el juego responda a las entradas del usuario y muestre mensajes adecuados según si el número adivinado es correcto o no.
  • Historial de Intentos: Crear un historial para que los jugadores vean sus intentos anteriores, añadiendo un elemento de seguimiento al juego.
  • Diseño Responsivo: Asegurar que la interfaz se vea bien en diferentes dispositivos, lo que se logró utilizando un framework CSS ligero.

Casa de Venta de Especias Sierra Morena

Sierra Morena es una tienda en línea dedicada a la promoción y comercialización de productos naturales de alta calidad.

Lenguajes Utilizados:

  • ✔️ HTML5 y CSS3: Para la estructura y el diseño de la página web.
  • ✔️ Bootstrap 5: Para asegurar que el sitio sea responsivo y tenga un diseño atractivo y moderno.
  • ✔️ JavaScript: Para mejorar la interactividad del sitio, como la funcionalidad de los botones de compra.

Desafíos Superados:

  • Diseño Responsivo: Asegurarse de que la tienda se vea bien en dispositivos móviles y de escritorio fue un reto, pero gracias a Bootstrap, logramos una interfaz amigable.
  • Optimización de Carga: Trabajar en la optimización de imágenes y recursos para que la página cargue rápidamente, mejorando la experiencia del usuario.
  • Integración de Productos: La organización y presentación de una amplia variedad de productos de manera clara y atractiva fue un desafío, pero logramos crear un diseño que facilita la navegación.
